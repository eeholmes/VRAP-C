• Run VRAPS tab: Use this tab to select a rav file and run VRAPS on it.
    • Step 1: Input data
      Select a data input method, one of:
      • Upload rav file
        • Click the "Browse..." button immediately beneath "Select rav file".
        • When you select a file from your local computer, it will be uploaded.
      • Choose a demo file
        • In the "Select demo file" list, choose one of the example rav files.
    • Optional Step 1b: Edit the rav file parameters.
      • Select the "Edit Rav file" tab.
      • Modify desired parameter values.
      • Click "Save edits".
      • Return to the "Run VRAPS" tab and resume with Step 2. When "Run VRAPS" is clicked, the simulation will use the modified parameter values.
        Rav parameters can be edited and run as many times as desired.
        • The rav editor is not displayed unless a rav file has been selected.
    • Step 2: Choose the number of runs per simulation
      • Select the number of runs or use the NRuns value in the rav file.
        When you change your selection, the application will calculate a new estimate of the simulation run time.
    • Step 3: Run VRAPS
      • Click the "Run VRAPS" button. A calculation-in-progress display will appear until the run completes.
      • When the run is finished, VRAPS will display the "Results (.sum)" summary tab.
      • To adjust parameters, run another simulation, or download output files or demos, return to the "Run Vrap" tab.
        • The "Run VRAPS" button is not displayed unless a rav file has been selected.
    • View results
      • When the simulation completes, VRAPS displays the summary "Results(.sum)" tab. To see other simulation results, select the appropriate Results tab as described below.
    • Download VRAPS output files
      • To download a report or other output file or, if the input rav file was modified, a rav file containing the parameter values for which the simulation was run, open the "Downloads" tab and click the appropriate download buttons.
        Before you download text files, select your operating system from the "Choose OS" list in order to download a file formatted appropriately for your platform.
        Downloads are only available after a simulation has been run.
    • Download VRAPS demo files
      • To download a copy of one of the VRAPS demo files, open the "Downloads" tab, select the desired demo file in the "Download the example rav files" list, then click the "Download Example file" button.
  • Edit Rav file tab : In this tab you can modify the parameters of the selected rav file before running VRAPS.
    • In the editor, modify rav parameters as desired.
      This affects only the parameters used in the VRAPS simulation — the original file is not changed.
    • Once you've finished making changes, click "Save edits".
      This saves a temporary copy of your modified file within the app only. To create a local copy of the file, either select the editor's contents and copy and paste to a local file on your computer or, once the simulation has run, download a copy from the "Downloads" tab (see below).
    • Return to the "Run VRAPS" tab and click the "Run VRAPS" button to run a simulation incorporating the modified parameters.
    • Parameters can be edited and re-run as many times as desired. Names of downloadable, modified rav files containing the edited parameters include the original rav file name, the edit version, and a timestamp.
    • The rav editor is reloaded when a new input file is selected.
  • Results tabs
      When a simulation has been completed, the Results tabs display the generated results. If no simulation has been run since the most recent file upload, the Results tabs contain brief help messages.
      When the input source selection changes or a new file is uploaded, any existing results displays are deleted and the Results tabs will display help messages.
    • Results(.sum) displays a summary of current simulation results.
      • #.#BYRSTART
    • Results(.byr) displays estimated bycatch rate results.
      • #.#BYREND
    • Results(.esc) displays estimtaed escapement results.
  • Downloads tab When a simulation has been run, results and demo rav files can be downloaded via the Downloads tab.
    The Downloads tab is cleared if the input source or demo file selection is changed or if a new rav file is uploaded.
    • Report : A PDF file containing a description of the model and results as well as the summary data from the simulation and a listing of a rav file containing the simulation's input parameters.
      • #.#BYRSTART
    • Byr file : A text file containing bycatch rate estimates.
      • #.#BYREND
    • Esc file : A text file containing escapement estimates.
    • Sum file : A text file containing summary data.
    • Rav : A rav file containing the parameter settings used during the simulation.
      Available only for runs based on edited parameter sets.
    • OS type : Specifying the OS type allows VRAPS to format text files appropriately for different platforms.
    • Demo files : Select the desired file in the "Download the example rav files" list and click "Download Example file".
