Update Rav

Rav parameters from an input file can be edited in this tab:

  • Select the Run VRAP tab.
  • Select a source of input data.
  • Either upload a data file or select a demo file, depending on the input data source chosen.
  • Select this tab. Modify desired parameters and click Save edits.
  • In the Run VRAP tab, click the Run VRAP button to run a model incorporating your parameter changes.

For more information, select the Help tab.
