• Run VRAP tab: Click here to upload a rav file and run VRAP.
    • Step 1: Input rav file
      Select a data input method:
      • Upload rav file
        • Click the "Browse..." button immediately beneath "Choose rav file from your file system".
        • When you select a file from your local computer, it will be uploaded.
      • Choose a demo rav file
        • In the "Select demo file" list, choose one of the available demos.
    • Optional Step 1b: Edit the input parameters.
      • Select the "Update Rav" tab.
      • Modify desired parameter values.
      • Click "Save edits".
      • Return to the "Run VRAP" tab and resume with Step 2. When "Run VRAP" is clicked, the simulation will run using the modified parameter values.
        Edited parameter sets can be re-edited and re-run as many times as desired.
    • Step 2: Choose number of runs per simulation
      • Select the number of runs for the simulation, or use the number of runs embedded in the rav file.
        When you change your selection, the application will calculate a new estimate of the simulation run time.
    • Step 3: Run VRAP
      • Click the "Run VRAP" button. The application will switch to the "Results" tab and display a calculation-in-progress screen until the simulation run completes.
        To adjust parameters, run another simulation, or download output files or demos, return to the "Run Vrap" tab.
    • View results
      • When the simulation completes, VRAP will display the summary "Results(.sum)" tab. To view other simulation results, select the appropriate Results tab, as described below.
    • Download VRAP output files
      • To download a report, bycatch rate or escapement estimates, a summary of the simulation, or (for edited ravs only) a rav file corresponding to the most recent simulation run, open the "Downloads" tab and click the appropriate download buttons. .
        Before you download, select your operating system from the "Choose OS" list in order to download a file formatted appropriately for your platform.
        Downloads are only available after a simulation has been run.
    • Download VRAP demo files
      • To download a copy of one of the VRAP demo files, open the "Downloads" tab, select the desired demo file in the "Download the example rav files" list, then click the "Download Example file" button.
  • Update Rav tab
    • Permits editing parameters from uploaded or demo rav files for input into VRAP.
    • Update the desired parameters, then click "Save edits".
    • Return to the "Run VRAP" tab and click "Run VRAP" to run a simulation incorporating the modified parameters.
    • Parameters can be re-edited and re-run as many times as desired. Names of downloadable, modified rav files containing the edited parameters (see "Downloads" below) contain the original rav file name, the edit version, and a timestamp.
    • The rav editor is reloaded when a new input file is selected.
  • Results tabs
      When a simulation has been run, the Results tabs display the output values.
      Results tabs are cleared if the input source or demo file selection is changed or if a new file is uploaded.
    • Results(.sum) displays a summary of current simulation results.
    • Results(.byr) displays estimated bycatch rate results.
    • Results(.esc) displays estimtaed escapement results.
  • Downloads tab When a simulation has been run, results and demo rav files can be downloaded from the Downloads tab.
    The Downloads tab is cleared if the input source or demo file selection is changed or if a new file is uploaded.
    • Report : a PDF file containing a description of the model and results as well as the summary data from the simulation and a rav file listing reflecting the simulation input parameters.
    • Byr file : a text file containing bycatch rate estimates.
    • Esc file : a text file containing escapement estimates.
    • Sum file : a text file containing summary data.
    • Rav : a (text) rav file containing the parameter settings used during the simulation. Available only for runs based on edited parameter sets.
    • OS type : specifying the OS type allows VRAP to format text files appropriately for different platforms.
    • Demo files : Select the desired file in the "Download the example rav files" list and click "Download Example file".
