VRAP Downloads

VRAP output and demo rav files can be downloaded from this tab:

  • Select the Run VRAP tab.
  • Select a source of input data.
  • Either upload a data file or select a demo file, depending on the chosen input data source.
  • Click the Run VRAP button and wait for the simulation to finish.

If you change the input data source, upload a new data file, or select a new demo file, the output and demo files will no longer be available for downloading until VRAP is run again.

For more information, select the Help tab.
