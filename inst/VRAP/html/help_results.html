VRAP Results

VRAP results are available in this tab once you follow these steps:

  • Select the Run VRAP tab.
  • Select a source of input data.
  • Either upload a data file or select a demo file, depending on the chosen input data source.
  • Click the Run VRAP button and wait for the simulation to finish.

If you change the input data source, upload a new data file, or select a new demo file, the results display will be cleared.

For more information, select the Help tab.
